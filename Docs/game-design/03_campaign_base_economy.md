# Warzone 战略层、基地与经济方案

## 1. 战略地图

战略地图表现为连续世界感，但系统上以区域、道路、地点、基地、前哨、事件和任务节点组织。玩家看到的是一个正在变化的后末日区域，而实现上不需要完整开放世界自由漫游。

地图上的医院、仓库、加油站、农场、工厂、学校、商场、道路关口、巢穴、废弃营地等兴趣点都应有长期状态。它们可以被探索、搜刮、清理、重新恶化、资源减少、变成前哨或基地候选点。世界不需要明确“玩家控制区域”的涂色机制，区域主要通过威胁等级、资源状态、道路通行、前哨网络和任务机会来表达。

## 2. 行动驱动时间

时间按行为推进。旅行、修复、训练、建设、任务、休息、交易、搬迁和处理伤员会消耗时间。玩家停在战略地图界面不动时，世界不自然流逝。

主线一般允许玩家准备很久再做，不做强硬时限。部分事件和动态任务可以随行动时间变化，但核心推进不应被倒计时压迫。战略压力来自资源和机会，而不是 UI 上的倒计时。

## 3. 区域资源衰减

区域资源采用类似补给速率逐步衰减的模型。第一次或早期进入某地点，收益等级较高；频繁搜刮后收益下降。经过一段行动时间后，部分基础资源可以恢复，但高价值资源不会无限高产。

资源衰减应表现为世界被利用，而不是简单惩罚。地点可以从“富集”变成“常规”，再变成“贫瘠”。前哨、清巢、贸易、道路恢复或特殊事件可以改善资源获取，但不能让玩家在新手区域永久高速刷资源。

## 4. 基地

基地由地图中的现实地点清理转化而来。玩家先进入一个可作为基地的地点，清理敌人，完成基础修整，然后该地点成为无敌对单位的安全场景。

基地的本质是基建系统的接入点，也是玩家发展成果的展示空间。它不采用自由摆放建筑，不做细粒度装修，不做完整居民生活模拟。基地采用类似《腐烂国度》的栏位与模块系统。不同基地有不同栏位数量、位置优势、防御条件、周边资源、特殊功能和迁移价值。

基地模块包括医疗、武器维护、车辆维修、仓储、训练、生活保障、情报、贸易、防御、改造素材处理等。基地本身决定栏位上限，升级决定效率和功能。模块可以消耗持续后勤，但首版不应让日常维护过于繁琐。

## 5. 前哨

前哨是小型据点，不能替代主基地。它们提供单一或少量功能，例如临时补给、仓储、休整、区域资源采集、简单维修、情报或威胁监控。前哨的意义在于扩大战略半径、降低行动成本、提供中转和维持区域活动。

## 6. 资源结构

资源分为三类：后勤资源包、可用物品、耐久资产。

后勤资源包包括食物/生活物资、医疗资源、通用弹药补给、燃料、建材、改造素材和货币/交易凭证等。它们用于基地维持、维修、补员、建设、交易和长期行动。

可用物品包括医疗包、烟雾弹、手雷、震爆、维修工具、夜视/照明工具、解毒或抗感染药物等。它们可以被带入战场并在局内直接使用。

耐久资产包括成员、武器、护甲、车辆和重要装备。它们会遗失、损坏、回收或长期保留。当前原则是不做普通武器护甲长期耐久，主要处理战斗中损坏、遗失和回收。

## 7. 商人和交易

商人系统主要承担情报、特殊任务、特殊交易和稀有装备供应。基础兜底也需要存在，但不应成为无限补给源。商人可以高价出售重武器、载具、改造材料、任务线索或特殊物品，部分交易需要货币加实物资源。

## 8. 主线与动态任务

主线承担故事推进、区域开放、世界状态变化、敌人生态变化、新武器/载具类别、新改造能力、新任务类型和势力关系变化。主线不是单纯剧情播片，也不是技术树任务。每次主线推进最好同时打开机会和制造风险。

动态任务主要提供资源维持、局部机会和风险选择。动态任务收益会递减，不能让玩家靠反复刷支线直接跳过主线。高级装备和关键改造应主要来自主线、高危地点、夜间敌人、精英/Boss、特殊商人和基地处理，而不是普通重复任务稳定产出。

## 9. M9 最小结算骨架

M9 不是完整基地经济，只是把战斗结果写回长期状态：

- 角色死亡会写回 Campaign 的成员状态
- 撤离成员会保留可用状态或改为伤员状态，按结算规则处理
- 搜索得到的战利品会进入 Campaign 库存
- 地点清理、搜索完成和威胁变化会写回 Site 状态
- MissionHistory 记录一次任务结果，供后续系统查询

本轮原则是先把长期状态闭环跑通，再扩展经济、商人、制造和正式存档。
