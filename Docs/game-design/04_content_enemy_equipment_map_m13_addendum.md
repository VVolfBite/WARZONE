# M13 Demo Catalog Addendum

This addendum records the canonical demo content pack used for code-only validation.

## Demo Catalog Contents

- weapons: `rifle`, `pistol`, `enemy_claws`
- enemies: `infected_basic`, `infected_runner`, `hostile_scavenger`
- resources: `food`, `medicine`, `ammo`, `fuel`, `building_material`, `modification_material`, `generic_loot`
- items: `medical_supplies`, `ammunition_box`, `scrap_material`
- sites: `medical_clinic`, `supply_depot`, `residential_block`, `industrial_yard`
- outposts: `basic_outpost`, `watch_post`
- loot profiles: `medical_loot`, `supply_loot`, `generic_loot`
- environment zones: smoke, fire, toxic, light, darkness
- vision equipment: `basic_night_vision`, `flashlight`
- base modules: `storage`, `infirmary`, `workshop`, `training`, `watchtower`

## Intent

The demo catalog is a seed set for tests, save/load, mission preparation, and campaign startup.
It is not a complete content library and it is not a Unity authoring pipeline.
