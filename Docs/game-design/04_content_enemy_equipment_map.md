# Warzone 敌人、装备与地图内容框架

## 1. 敌人生态

首发敌对生态包括感染群体、变异动物、感染植物/环境体和人类敌对势力。具体阵营关系后置，但底层必须支持阵营关系表。不能把“所有敌人永远互不攻击”写死在单位逻辑里。

敌人强度由区域等级、主线阶段、昼夜、地图环境共同决定，不根据玩家战力动态同步缩放。玩家变强后应能明显感受到成长价值，但更危险区域和夜战会引入更复杂、更危险的组合。

首批敌人职责优先包括：数量压力、装甲/耐久压力、远程威胁、区域控制、Boss/精英。突击、隐蔽伏击、召唤型可以后续加入。

## 2. 巢穴与威胁源

普通巢穴可进入、可清理，地图外仍保留更深层不可触及的威胁源。清理巢穴可以降低地图敌人补充速度、降低区域整体威胁、解锁资源/道路/前哨/特殊奖励，但不应永久清空世界。

敌人从地图外不可进入区域进入战场。这一圈外部区域可视为抽象威胁源，不需要完整模拟，但敌人进入地图时必须沿可理解的路线出现，例如道路、树林边缘、地下通道、建筑群或巢穴方向。

## 3. 敌人奖励

人类敌人或普通感染者可以掉落简单战斗物品。感染者、变异体和夜间敌人可以掉落改造素材，例如肉体强化素材、特殊组织、感染样本等。Boss 房间、特殊场景和精英地点提供更具体、更高价值的奖励。

不能让刷普通敌人成为主要资源来源。资源主要来自地点搜索、高危区域、任务目标和特殊敌人。

## 4. 武器类别

完整游戏可覆盖突击步枪/半自动步枪、冲锋枪、霰弹枪、轻机枪/重机枪、狙击步枪/精确射手步枪、榴弹发射器、火箭筒/反装甲武器、喷火器/燃烧武器、近战武器和手枪。

但首版不能把完整武器图鉴当作范围。POC 只需要少数战术角色明确的武器：步枪、机枪、霰弹枪、精确射手/狙击、爆炸武器、烟雾和医疗。武器差异以战术功能为主，例如压制、清群、破甲、破门、燃烧、反载具、远程精确，而不是只看 DPS。

## 5. 改装与弹药

武器改装采用简单插槽，例如瞄具、弹匣、枪口和挂件，不做深度枪械拼装器。弹药按大类共享，例如步枪弹、手枪弹、霰弹、重武器弹，而不是每把枪独立弹药。

护甲采用等级加功能特性。轻、中、重护甲决定基础防护；防火、防毒、夜视、携行、抗压等功能作为扩展特性，但首版不宜过多。

## 6. 夜战

夜战是核心风险层，应早做。夜战影响视野、敌人强度和收益。部分特殊材料只能从夜间敌人或夜晚场景获得，类似“高风险夜魔掉改造资源”的设计。夜战装备不是硬门槛，但能明显降低风险；玩家可以不带夜战装备行动，只是视野、识别和撤离压力更大。

## 7. 天气

首版天气只做少量静态环境状态，例如雾、雨、强风。天气主要服务视觉氛围、视野和少量简单修正，不做复杂动态天气系统。

## 8. 地图内容生产边界

普通地图以模块化/程序化为主，特殊手工地图承担主线、大型建筑和记忆点。大型商场、医院、地铁站、工厂、游乐园、军事设施可以有更复杂结构和特殊机制；普通房屋和街区使用简化模板与战术节点。

地图生成器的目标不是生成完整城市，而是生成可进入、可识别、可重复进入、带有资源梯度和威胁方向的小型连续区域。
